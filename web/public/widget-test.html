Rent This Dumpster

This page simulates a tenant's existing website. The floating "Book Now" button in the bottom-right corner is the RentThisApp widget.

Test Controls

Open the widget programmatically:

Embed Code

Add this to any webpage:

<script src="https://serviceos-web-zeta.vercel.app/widget.js" data-slug="rent-this-dumpster"></script>

Optional Attributes

  • data-position="bottom-left" — Move button to bottom-left
  • data-button-text="Get a Quote" — Custom button text
  • data-auto-open="true" — Open modal immediately

Booking Callback

Listen for completed bookings:

window.rentThisAppOnBooking = function(data) {
  console.log('Booking completed:', data);
  // Track conversion, redirect, etc.
};
